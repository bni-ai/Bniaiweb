# BNI 平台架構對焦圖

> 最後更新：2026-06-05
> 策略：先建工具系統（B），知識庫內容後補（C）

---

## 核心目標

讓一個從未參加過 BNI 的新會員，使用系統就能從不懂到懂、做好所有事。

---

## 三大工具系統（優先開發）

### 工具 1：引薦單系統

**現有資料：**
- `docs/02-member-system.md`：引薦等級 1-5 定義、填寫要點、24 小時跟進規範
- `其它/華AI會員引薦資訊表.csv`：真實範本

**功能範圍：**
- 填寫：輸入姓名 / 電話 / 需求痛點 / 引薦等級
- 發送：通知被引薦會員 + 被推薦的目標客戶
- 追蹤：24 小時提醒、進度回報（打了嗎？約了嗎？成交了嗎？）
- TYFCB：成交後申報金額、感謝訊息
- 統計：本月給出 / 收到的引薦數

**AI 功能：**
- 「你說的這個朋友可能適合推薦給 [XX 夥伴]，要填一張嗎？」

---

### 工具 2：一對一管理系統

**現有資料：**
- `docs/bni-one-on-one-form-skill.md`：AI 表單模板，10 個區塊，完整 HTML 輸出
- Training 逐字稿（41 分鐘）：如何進行、常見問題
- `其它/` 多份真實一對一 PDF & 文字版（老魚、Sean、Alex 等）

**功能範圍：**
- 預約：找時間、發邀請、確認地點
- 備課：事前填寫 GAINS 表（AI 引導對話，一次一區塊）
- 會中：對方的 GAINS 表可查看 + 做筆記
- 會後：記錄結論、引薦動作、下次跟進提醒
- 統計：本週做了幾次、與誰還沒做過

**AI 功能：**
- 「你和 [XX] 的一對一已經 6 週沒做了，要預約嗎？」
- 「根據你的描述，你的業務人脈圈裡該找 [ABC 行業]」

---

### 工具 3：整合溝通（取代 LINE + Zoom + Google Meet）

**現有資料：**
- `docs/line-works-alternatives/`：替代方案調研
- `docs/workflow.md`：Zoom 逐字稿 → 腳本 → 自動推播的現有流程

**功能範圍：**
- 公告：取代 LINE 群組 — 週報發布、緊急通知
- 例會通知：出席確認、行前提醒
- 視訊連結：一對一 / 例會線上連結整合
- 例會紀錄：逐字稿 → 自動摘要 → 引薦單自動產生

**AI 功能：**
- 「本週例會逐字稿已分析，偵測到 3 個潛在引薦機會：...」
