# BNI 華AI分會管理平台

BNI ACCELERATE 華AI分會的會員管理與簡報自動化系統。

## 功能

- 會員資料管理（Gmail 登入）
- 每週簡報自動產出
- 8 分鐘短講管理
- 一對一預約系統
- 活動管理
- 培訓追蹤
- AI 助手

## 技術棧

- Next.js
- Supabase（資料庫 + 認證）
- Tailwind CSS
- Vercel（部署）
